# Product Overview

## Project: Couple Fitness Check-in Mini Program

A WeChat mini-program designed for couples to track fitness activities together, with gamified engagement features and emotional connection building.

### Core Value Proposition
- **3-second check-in**: Minimal friction for daily fitness logging
- **Couple interaction**: Strengthens emotional bonds through shared fitness goals
- **Gamification**: Virtual pet raising, level systems, and achievement unlocking
- **Memory creation**: Long-term relationship milestones and shared accomplishments

### Key Features (MVP)
- User login and couple pairing
- Quick fitness check-in (exercise type, duration, date)
- Shared calendar view of both partners' activities
- Basic messaging between partners
- Achievement/medal system

### Future Features (V1.0+)
- Virtual pet companion that grows with couple's fitness
- Bet/wager system for motivation
- Level progression system
- Photo collage creation
- Energy bank system
- Mystery box rewards

### Target Users
- Couples aged 18-40
- Fitness-motivated individuals
- Users seeking relationship engagement tools
- WeChat ecosystem users (primarily China)

### Technical Deployment
- **Frontend**: WeChat mini-program (native)
- **Backend**: RuoYi-Vue (Spring Boot 2.5.15)
- **Database**: MySQL 8.0+
- **Authentication**: JWT tokens
