## 修复的编译错误

### 错误 1: CoupleUserController.java - uploadFile 方法不存在
**错误信息**:
```
找不到符号: 方法 uploadFile(org.springframework.web.multipart.MultipartFile)
位置: 类型为com.ruoyi.web.service.IFileUploadService的变量 fileUploadService
```

**原因**: `FileUploadServiceImpl` 只有 `uploadCheckInPhoto` 方法，没有 `uploadFile` 方法

**修复方案**:
```java
// 修改前
String avatarUrl = fileUploadService.uploadFile(file);
return success("头像上传成功", avatarUrl);

// 修改后
String avatarUrl = fileUploadService.uploadCheckInPhoto(file);
return AjaxResult.success("头像上传成功", avatarUrl);
```

**文件**: `RuoYi-Vue/ruoyi-admin/src/main/java/com/ruoyi/web/controller/CoupleUserController.java`

### 错误 2: ExerciseReminderTask.java - getOpenId() 方法不存在
**错误信息**:
```
找不到符号: 方法 getOpenId()
位置: 类型为com.ruoyi.web.domain.CoupleUser的变量 user
```

**原因**: `CoupleUser` 实体类没有 `openId` 字段，只有 `wechatId` 字段

**修复方案**:
```java
// 修改前
if (user == null || user.getOpenId() == null) {
    log.warn("用户不存在或openId为空: userId={}", setting.getUserId());
    continue;
}
boolean success = weChatNotificationService.sendExerciseReminder(
    user.getOpenId(),
    setting.getReminderTime()
);

// 修改后
if (user == null || user.getWechatId() == null) {
    log.warn("用户不存在或wechatId为空: userId={}", setting.getUserId());
    continue;
}
boolean success = weChatNotificationService.sendExerciseReminder(
    user.getWechatId(),
    setting.getReminderTime()
);
```

**文件**: `RuoYi-Vue/ruoyi-admin/src/main/java/com/ruoyi/web/task/ExerciseReminderTask.java`
