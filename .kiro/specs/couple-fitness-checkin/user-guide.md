# 情侣健身打卡小程序 - 用户使用文档

## 📱 欢迎使用情侣健身打卡小程序

这是一款专为情侣设计的健身打卡工具，帮助你和伴侣一起记录运动、互相激励、共同进步。

### 1. 首次使用

#### 1.1 打开小程序
1. 在微信中搜索"情侣健身打卡"小程序
2. 点击进入小程序
3. 首次使用需要授权微信登录

#### 1.2 授权登录
1. 点击"微信授权登录"按钮
2. 确认授权获取微信昵称和头像
3. 登录成功后自动进入主页

> **提示**：小程序会记住你的登录状态，下次打开自动登录

## 💑 情侣配对

### 2.1 为什么要配对？
配对后，你可以：
- 查看伴侣的打卡记录
- 对比双方的运动数据
- 为对方的打卡点赞和评论
- 接收伴侣的打卡通知
- 一起完成情侣目标

### 2.2 如何配对？

#### 方式一：生成邀请码（推荐）
1. 进入"个人中心"页面
2. 点击"情侣配对"
3. 点击"生成邀请码"
4. 将6位数字邀请码发送给伴侣
5. 或让伴侣扫描二维码

#### 方式二：输入邀请码
1. 进入"个人中心"页面
2. 点击"情侣配对"
3. 输入伴侣的6位邀请码
4. 点击"发送配对请求"
5. 等待对方接受

#### 2.3 接受配对请求
1. 收到配对请求通知
2. 进入"个人中心" → "情侣配对"
3. 查看配对请求
4. 点击"接受"完成配对

> **注意**：每个用户只能与一个伴侣配对

1. 进入"个人中心" → "情侣配对"
2. 点击"解除配对"
3. 确认操作（此操作不可撤销）

## 🏃 快速打卡

### 3.1 创建打卡记录

#### 步骤1：进入打卡页面
- 点击主页的"快速打卡"按钮
- 或点击底部导航栏的"打卡"图标

#### 步骤2：上传照片
1. 点击"拍照"或"从相册选择"
2. 选择一张运动照片
3. 照片会自动压缩至2MB以内

> **提示**：照片可以是自拍、运动场景、健身器材等

#### 步骤3：选择运动类型
从以下类型中选择：
- 🏠 **居家运动**：在家做的运动（瑜伽、健身操等）
- 🏋️ **健身房**：在健身房的力量训练、器械训练
- 🌳 **户外运动**：户外跑步、骑行、爬山等
- 🏃 **跑步**：专门的跑步训练
- 🧘 **瑜伽**：瑜伽练习
- 💪 **力量训练**：专门的力量训练

#### 步骤4：输入运动时长
- 输入运动时长（1-300分钟）
- 系统会自动计算卡路里消耗

#### 步骤5：提交打卡
- 点击"提交打卡"按钮
- 等待上传完成
- 看到成功提示即完成打卡

### 3.2 卡路里计算说明

系统根据以下因素自动计算卡路里：
- 运动类型（不同运动的MET值不同）
- 运动时长
- 你的体重（在个人信息中设置）

**MET值参考**：
- 居家轻运动：3.0
- 瑜伽：3.5
- 跑步(8km/h)：8.0
- 健身房力量训练：6.0
- 户外运动：5.0

**计算公式**：
```
卡路里 = (MET × 体重kg × 时长分钟) / 60
```

**示例**：
体重60kg，跑步30分钟
卡路里 = (8.0 × 60 × 30) / 60 = 240卡路里

### 3.3 离线打卡

如果网络不稳定：
1. 正常填写打卡信息并提交
2. 数据会自动保存在本地
3. 网络恢复后自动上传
4. 上传成功后会收到通知

## 🔒 隐私与安全

### 12.1 隐私设置

#### 数据共享设置
1. 进入"个人中心" → "隐私设置"
2. 选择数据共享级别：
   - **完全共享**：伴侣可以查看所有详细数据
   - **基础共享**：伴侣只能查看基础统计（打卡次数、总时长）
   - **不共享**：伴侣无法查看你的数据
3. 点击"保存"

### 12.2 数据安全

#### 数据加密
- 所有数据传输使用HTTPS加密
- 敏感信息加密存储
- 只有配对的伴侣可以查看你的数据

#### 数据备份
- 所有数据自动备份到云端
- 更换设备后可以恢复数据
- 数据保存期限：永久

### 12.3 账号注销

如果要注销账号：
1. 进入"个人中心" → "隐私设置"
2. 点击"注销账号"
3. 阅读注销说明
4. 确认注销（此操作不可撤销）

> **警告**：注销账号后，所有数据将被永久删除，包括：
> - 所有打卡记录
> - 所有成就徽章
> - 所有评论和互动
> - 情侣配对关系

## ❓ 常见问题

### 14.1 登录问题

**Q: 无法登录怎么办？**
A: 
1. 检查网络连接
2. 确认已授权微信登录
3. 尝试重新打开小程序
4. 如果仍无法登录，联系客服

**Q: 登录后数据丢失？**
A: 
1. 检查是否使用了正确的微信账号
2. 数据会自动从云端恢复
3. 如果数据仍未恢复，联系客服

### 14.2 配对问题

**Q: 邀请码无效？**
A: 
1. 确认邀请码是6位数字
2. 检查是否输入错误
3. 邀请码可能已过期，请重新生成

**Q: 无法配对？**
A: 
1. 确认双方都已登录
2. 确认对方已接受配对请求
3. 确认双方都没有其他配对关系

**Q: 如何更换伴侣？**
A: 
1. 先解除当前配对关系
2. 然后与新伴侣配对
3. 注意：解除配对不会删除历史数据

### 14.3 打卡问题

**Q: 照片上传失败？**
A: 
1. 检查网络连接
2. 确认照片大小不超过10MB
3. 尝试压缩照片后重新上传
4. 如果仍失败，尝试更换照片

**Q: 卡路里计算不准确？**
A: 
1. 确认已设置正确的体重
2. 卡路里计算基于标准MET值
3. 实际消耗可能因人而异

**Q: 可以补打卡吗？**
A: 
可以。在打卡页面选择过去的日期即可补打卡。

**Q: 可以修改已提交的打卡吗？**
A: 
不可以。如果信息有误，可以删除后重新打卡。

### 14.4 互动问题

**Q: 为什么看不到伴侣的打卡？**
A: 
1. 确认已完成配对
2. 确认伴侣已打卡
3. 尝试下拉刷新页面
4. 检查伴侣的隐私设置

**Q: 评论发送失败？**
A: 
1. 检查网络连接
2. 确认评论内容不超过200字
3. 确认评论不包含敏感词
4. 尝试重新发送

### 14.5 数据问题

**Q: 数据不同步？**
A: 
1. 检查网络连接
2. 下拉刷新页面
3. 等待自动同步完成
4. 如果仍不同步，联系客服

**Q: 连续打卡天数计算错误？**
A: 
1. 连续打卡要求每天至少打卡一次
2. 如果某天未打卡，连续天数会重置
3. 查看日历确认是否有遗漏

**Q: 可以导出数据吗？**
A:
