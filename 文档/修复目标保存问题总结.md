# 修复目标保存问题总结

## 问题描述

用户在保存目标时遇到两个错误：

1. **userId 为 null 错误**：`MethodArgumentNotValidException: Field 'userId': rejected value [null]`
2. **start_date 缺失错误**：`SQLException: Field 'start_date' doesn't have a default value`

## 问题根源

### 问题 1：userId 验证失败

- `Goal` 实体类的 `userId` 字段有 `@NotNull` 验证注解
- Spring 的 `@Validated` 在 Controller 方法执行前就进行验证
- 前端没有传递 `userId`（这是正确的，应该由后端从 JWT token 获取）
- 验证失败导致 Controller 中设置 userId 的代码根本没有执行

### 问题 2：数据库字段缺失

- 数据库表 `goal` 中 `start_date` 和 `end_date` 字段定义为 `NOT NULL`
- 插入语句中没有提供这些字段的值
- MySQL 报错：字段没有默认值

## 解决方案

### 修复 1：移除 userId 的 @NotNull 注解

**文件**：`RuoYi-Vue/ruoyi-admin/src/main/java/com/ruoyi/web/domain/Goal.java`

```java
// 修改前
@NotNull(message = "用户ID不能为空")
private Long userId;

// 修改后
/** 用户ID（由后端从JWT token中获取，不需要前端传递） */
private Long userId;
```

**原因**：
- userId 应该由后端从 JWT token 中获取，而不是由前端传递
- 这符合安全最佳实践，防止用户伪造他人的 userId
- Controller 中已经有完善的 userId 验证逻辑

#### 2.2 在 Service 层自动计算日期范围

**文件**：`RuoYi-Vue/ruoyi-admin/src/main/java/com/ruoyi/web/service/impl/GoalServiceImpl.java`

添加了 `setGoalDateRange()` 方法：

```java
/**
 * 设置目标的开始和结束日期
 * 周目标：本周一到本周日
 * 月目标：本月1号到本月最后一天
 */
private void setGoalDateRange(Goal goal, String goalType) {
    java.util.Calendar calendar = java.util.Calendar.getInstance();
    Date now = new Date();
    calendar.setTime(now);

    if (GOAL_TYPE_WEEKLY.equals(goalType)) {
        // 周目标：本周一到本周日
        int dayOfWeek = calendar.get(java.util.Calendar.DAY_OF_WEEK);
        int daysToMonday = (dayOfWeek == java.util.Calendar.SUNDAY) ? -6 : (2 - dayOfWeek);
        calendar.add(java.util.Calendar.DAY_OF_MONTH, daysToMonday);
        Date startDate = calendar.getTime();
        
        calendar.add(java.util.Calendar.DAY_OF_MONTH, 6);
        Date endDate = calendar.getTime();
        
        goal.setStartDate(startDate);
        goal.setEndDate(endDate);
    } else if (GOAL_TYPE_MONTHLY.equals(goalType)) {
        // 月目标：本月1号到本月最后一天
        calendar.set(java.util.Calendar.DAY_OF_MONTH, 1);
        Date startDate = calendar.getTime();
        
        calendar.set(java.util.Calendar.DAY_OF_MONTH, 
            calendar.getActualMaximum(java.util.Calendar.DAY_OF_MONTH));
        Date endDate = calendar.getTime();
        
        goal.setStartDate(startDate);
        goal.setEndDate(endDate);
    }
}
```

在 `createGoal()` 方法中调用：
```java
// 设置目标的开始和结束日期
setGoalDateRange(goal, goalType);

// 设置初始状态
goal.setStatus("active");
goal.setCurrentValue(0);
```

## 注意事项

1. **数据库表结构**：确保 `goal` 表已经按照 `goal-table.sql` 创建
2. **JWT Token**：确保用户已登录，token 有效
3. **日期计算**：周目标从周一开始，月目标从1号开始
4. **软删除**：删除目标时只设置 `deleted_at`，不真正删除记录
5. **活跃目标**：一个用户同一类型只能有一个活跃目标
