# 实现对方打卡记录查看功能

## 功能说明

点击首页"对方的状态"区域，可以跳转到一个新页面查看对方的所有打卡记录，并可以进行点赞和评论。

## 实现内容

### 1. 修改首页

**文件**：`couple-fitness-weapp/pages/index/index.wxml`

**修改**：
- 给"对方的状态"区域添加 `bindtap="goToPartnerCheckIns"` 点击事件
- 添加"点击查看TA的打卡记录 >"提示文字
- 修改"去配对"按钮，添加 `catchtap` 阻止事件冒泡

**文件**：`couple-fitness-weapp/pages/index/index.js`

**添加方法**：
```javascript
goToPartnerCheckIns() {
  const { partnerInfo } = this.data;
  
  if (!partnerInfo) {
    wx.showToast({
      title: '还没有配对伴侣',
      icon: 'none'
    });
    return;
  }
  
  // 跳转到对方的打卡记录页面
  wx.navigateTo({
    url: `/pages/partner-checkins/index?partnerId=${partnerInfo.userId}&partnerName=${partnerInfo.nickname || '对方'}`
  });
}
```

**文件**：`couple-fitness-weapp/pages/index/index.wxss`

**添加样式**：
```css
.status-hint {
  margin-top: 12px;
  padding-top: 12px;
  border-top: 1px dashed var(--border-light);
  text-align: center;
}

.hint-text {
  font-size: 24rpx;
  color: var(--primary-color);
  font-weight: 500;
}
```

### 2. 创建对方打卡记录页面

**页面路径**：`pages/partner-checkins/index`

**文件列表**：
- `index.js` - 页面逻辑
- `index.wxml` - 页面结构
- `index.wxss` - 页面样式
- `index.json` - 页面配置

**主要功能**：
- 接收参数：`partnerId`（对方用户ID）和 `partnerName`（对方昵称）
- 加载对方的打卡记录列表
- 支持下拉刷新
- 支持上拉加载更多
- 显示点赞和评论按钮
- 可以给对方的打卡点赞和评论

### 3. 注册新页面

**文件**：`couple-fitness-weapp/app.json`

**添加**：
```json
"pages/partner-checkins/index"
```

## 使用流程

### 用户操作流程

1. **打开首页**
   - 查看"对方的状态"区域
   - 显示对方今天是否已打卡

2. **点击对方状态区域**
   - 跳转到对方的打卡记录页面
   - 页面标题显示"XXX的打卡记录"

3. **查看对方打卡记录**
   - 显示对方的所有打卡记录
   - 每条记录显示：
     - 运动类型
     - 运动时长
     - 消耗卡路里
     - 打卡照片（如果有）
     - 点赞和评论按钮

4. **互动操作**
   - 点击❤️按钮给对方点赞
   - 点击💬按钮给对方评论（开发中）
   - 下拉刷新列表
   - 上拉加载更多记录

## 技术实现细节

### 数据加载

**当前实现**（临时方案）：
```javascript
const response = await checkInAPI.getCheckInRecords({
  userId: partnerId,
  pageNum: pageNum,
  pageSize: pageSize
});
```

**问题**：
- 当前API可能不支持按userId查询
- 需要后端提供专门的接口

**建议后端添加接口**：
```java
@GetMapping("/user/{userId}")
public TableDataInfo getUserCheckIns(@PathVariable Long userId, 
                                     @RequestParam(defaultValue = "1") Integer pageNum,
                                     @RequestParam(defaultValue = "10") Integer pageSize) {
    startPage();
    List<CheckInRecord> list = checkInService.getCheckInsByUserId(userId);
    return getDataTable(list);
}
```
