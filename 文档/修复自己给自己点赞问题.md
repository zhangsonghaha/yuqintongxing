# 修复自己给自己点赞问题

## 完成时间
2026-03-11 23:30

## 问题发现

通过 MySQL MCP 工具查询数据库，发现了一个严重的业务逻辑问题：

**用户4342给自己的打卡记录点赞了！**

### 数据库证据

```sql
SELECT * FROM interaction WHERE type = 'like';
```

结果：
| interaction_id | record_id | user_id | type | created_at |
|----------------|-----------|---------|------|------------|
| 1 | 5 | 37 | like | 2026-03-11 22:04:16 |
| 2 | 6 | 37 | like | 2026-03-11 22:17:09 |

其中：
- **记录1**：用户37（用户4342）给记录5（用户37自己的打卡）点赞 ❌
- **记录2**：用户37（用户4342）给记录6（用户39/用户8260的打卡）点赞 ✅

## 问题影响

1. **违反业务逻辑**：自己不应该能给自己点赞
2. **数据不一致**：导致点赞统计不准确
3. **前端显示异常**：可能导致UI显示错误
4. **用户体验差**：用户可以刷自己的点赞数

## 解决方案

### 1. 删除错误数据 ✅

执行SQL删除错误的点赞记录：

```sql
DELETE FROM interaction 
WHERE interaction_id = 1 
  AND record_id = 5 
  AND user_id = 37 
  AND type = 'like';
```

**执行结果**：成功删除1条记录

**验证**：
```sql
SELECT * FROM interaction WHERE type = 'like';
```

现在只剩下正确的记录：
| interaction_id | record_id | user_id | 说明 |
|----------------|-----------|---------|------|
| 2 | 6 | 37 | 用户4342给用户8260点赞 ✅ |

### 2. 修改后端代码 ✅

**文件**：`RuoYi-Vue/ruoyi-admin/src/main/java/com/ruoyi/web/service/impl/InteractionServiceImpl.java`

**修改内容**：在 `likeCheckIn()` 方法中添加检查

**修改前**：
```java
@Override
@Transactional
public Map<String, Object> likeCheckIn(Long recordId, Long userId) {
    Map<String, Object> result = new HashMap<>();
    
    // 检查是否已点赞
    Interaction existingLike = interactionMapper.selectLikeByRecordIdAndUserId(recordId, userId);
    if (existingLike != null) {
        result.put("code", 400);
        result.put("msg", "已经点赞过了");
        return result;
    }
    
    // 创建点赞记录
    // ...
}
```

**修改后**：
```java
@Override
@Transactional
public Map<String, Object> likeCheckIn(Long recordId, Long userId) {
    Map<String, Object> result = new HashMap<>();
    
    // ✅ 获取打卡记录
    CheckInRecord record = checkInMapper.selectCheckInById(recordId);
    if (record == null) {
        result.put("code", 404);
        result.put("msg", "打卡记录不存在");
        return result;
    }
    
    // ✅ 检查是否是自己的记录
    if (record.getUserId().equals(userId)) {
        result.put("code", 400);
        result.put("msg", "不能给自己的打卡记录点赞");
        return result;
    }
    
    // 检查是否已点赞
    Interaction existingLike = interactionMapper.selectLikeByRecordIdAndUserId(recordId, userId);
    if (existingLike != null) {
        result.put("code", 400);
        result.put("msg", "已经点赞过了");
        return result;
    }
    
    // 创建点赞记录
    // ...
}
```
