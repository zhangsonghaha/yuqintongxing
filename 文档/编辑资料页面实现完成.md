# 编辑资料页面实现完成

## 修复的问题

### 1. WXML 编译错误
**问题**：微信小程序 WXML 不支持在模板表达式中直接调用 `.substring()` 方法
```
{{ (userInfo.nickname || '用户').substring(0, 1) }}  ❌ 错误
```

**解决方案**：在 JS 中预处理数据
```javascript
// 在 JS 中计算头像文字
const nickname = userInfo.nickname || '用户';
const avatarText = nickname.substring(0, 1).toUpperCase();

// 在 WXML 中直接使用
{{ avatarText }}  ✅ 正确
```

### 2. 默认头像图片不存在
**问题**：`/assets/default-avatar.png` 文件不存在，返回 500 错误

**解决方案**：使用纯 CSS 实现的头像占位符
- 显示昵称首字母
- 粉色渐变背景
- 圆形设计，与整体风格一致

### 3. 字体加载失败
**问题**：尝试加载阿里云 CDN 字体文件（`http://at.alicdn.com/t/c/font_xxx.woff2`）

**解决方案**：移除外部字体引用，使用系统字体

### 4. API 路径不匹配
**问题**：前端调用 `/couple/user/profile`，但后端接口是 `/api/user/profile`

**解决方案**：统一使用 `/api/user/profile` 路径

## API 接口

### 1. 获取用户信息
```
GET /api/user/profile
Authorization: Bearer {token}
```

**响应**：
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {
    "userId": 1,
    "nickname": "用户昵称",
    "avatar": "https://...",
    "gender": "男",
    "birthday": "1990-01-01",
    "signature": "个性签名"
  }
}
```

### 2. 更新用户信息
```
PUT /api/user/profile
Authorization: Bearer {token}
Content-Type: application/json

{
  "nickname": "新昵称",
  "gender": "女",
  "birthday": "1995-05-20",
  "signature": "新的个性签名"
}
```

**响应**：
```json
{
  "code": 200,
  "msg": "更新成功"
}
```

### 3. 上传头像
```
POST /api/user/avatar
Authorization: Bearer {token}
Content-Type: multipart/form-data

file: [图片文件]
```

**限制**：
- 文件大小：最大 5MB
- 文件格式：JPG、JPEG、PNG

**响应**：
```json
{
  "code": 200,
  "msg": "头像上传成功",
  "data": "https://example.com/uploads/avatar.jpg"
}
```

## 数据验证

### 前端验证
- 昵称：必填，最多20字符
- 性别：可选（男/女/保密）
- 生日：可选，不能晚于今天
- 个性签名：可选，最多100字符
- 头像：可选，最大5MB，仅支持 JPG/PNG

### 后端验证
- 昵称：敏感词过滤，长度限制
- 身高：100-250cm
- 体重：30-300kg
- 头像：文件大小和格式验证

## 使用说明

### 1. 进入编辑资料页面
从"我的"页面点击"编辑资料"菜单项

### 2. 修改头像
1. 点击头像区域
2. 选择相册或拍照

1. 填写表单字段
2. 点击"保存"按钮
3. 等待保存成功提示
4. 自动返回上一页

- 如果用户没有上传头像，显示昵称首字母
- 粉色渐变背景，白色文字
- 自动大写显示
