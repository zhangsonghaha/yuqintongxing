## 数据库变更

### 新增表

1. **user_reminder_setting** - 用户提醒设置表
   - setting_id: 设置ID
   - user_id: 用户ID
   - exercise_reminder_enabled: 运动提醒开关
   - reminder_time: 提醒时间
   - like_notification_enabled: 点赞通知开关
   - comment_notification_enabled: 评论通知开关
   - goal_notification_enabled: 目标通知开关

2. **privacy_setting** - 隐私设置表
   - setting_id: 设置ID
   - user_id: 用户ID
   - allow_partner_view_checkins: 伴侣查看打卡记录权限
   - allow_partner_view_stats: 伴侣查看统计数据权限
   - allow_partner_view_achievements: 伴侣查看成就权限
   - data_share_scope: 数据共享范围

## API 接口

### 通知设置
- GET `/api/reminder-setting` - 获取提醒设置
- PUT `/api/reminder-setting` - 更新提醒设置

### 用户信息
- GET `/api/user/profile` - 获取用户信息
- PUT `/api/user/profile` - 更新用户信息
- POST `/api/user/avatar` - 上传头像

### 隐私设置
- GET `/api/privacy` - 获取隐私设置
- PUT `/api/privacy` - 更新隐私设置
- POST `/api/privacy/deactivate` - 注销账号
- DELETE `/api/privacy/delete-all` - 删除所有数据
