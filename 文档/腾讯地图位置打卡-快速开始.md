## 🚀 快速部署（3步）

### 第1步：更新数据库
```bash
# 连接到 MySQL
mysql -u root -p

# 选择数据库
USE your_database_name;

# 执行迁移脚本
SOURCE RuoYi-Vue/sql/add_location_fields_to_checkin.sql;

# 验证表结构
DESCRIBE check_in_record;
```

### 第2步：重启后端服务
```bash
cd RuoYi-Vue/ruoyi-admin
mvn clean compile
mvn spring-boot:run
```

1. 打开微信开发者工具
2. 加载 `couple-fitness-weapp` 项目

## 🔧 故障排除

### 问题：位置获取失败
**解决方案**：
1. 检查 `app.json` 中的权限配置
2. 确认腾讯地图 API Key 正确
3. 查看微信开发者工具控制台错误

### 问题：地址解析失败
**解决方案**：
1. 检查网络连接
2. 确认腾讯地图 API 服务正常
3. 查看 API 调用返回的错误信息

### 问题：数据未保存
**解决方案**：
1. 检查数据库表结构是否更新
2. 查看后端日志中的 SQL 错误
3. 确认 MyBatis Mapper 配置正确
